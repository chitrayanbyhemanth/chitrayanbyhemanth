## ✨ Overview

**ChitraYan by Hemanth** is a premium, high-end photography portfolio designed to showcase cinematic wedding stories and emotional portraits. Based in Hyderabad, this digital experience reflects the elegance and raw emotion of the moments captured behind the lens.

The website emphasizes a **light luxury aesthetic**, utilizing smooth typography, fluid micro-animations, and a highly refined visual presentation.

## 🌟 Key Features

- **💎 Luxury Aesthetic:** Soft champagne accents, elegant typography (`EB Garamond` & `Cormorant Garamond`), and careful use of whitespace.
- **🎨 Custom Interactions:** Includes a beautifully crafted custom cursor (dot and expanding ring) that adapts to interactive elements.
- **🖼️ Cinematic Galleries:** Immersive grid layouts with stunning hover states that reveal content overlays, taglines, and subtle scaling animations.
- **📜 Scrolling Marquee:** A dynamic background scrolling text effect for the Services section that adds depth and movement.
- **📱 Fully Responsive:** Carefully constructed using modern CSS grids, flexbox, and clamp functions to look breathtaking on all devices.
